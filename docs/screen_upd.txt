Шаг 3: Полная перезапись slider_10_display.ino на U8g2 API.

Ключевые изменения:

    clearDisplay() → clearBuffer()
    display() → updateDisplay() (только изменённые тайлы!)
    setTextSize(N) → setFont(...) с подходящим шрифтом
    setCursor(x, y) → y теперь по baseline (нижний край текста), не по верхнему
    fillRect() → drawBox(), drawRect() → drawFrame()
    setTextColor(BLACK) → setDrawColor(0)
    Расчёт ширины текста → getStrWidth()
